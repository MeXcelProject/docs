Interfaces

API

MQL4 API functions. All methods are thread-safe. Any function call may throw APINotAvailable, so caller always must handle it properly. There is no way to handle event when API is about to become unavailable, but yet still available. It just suddenly dies, because of MQL4 imported functions call specifics (see FxDialogue.MeXcelTrader.MQL4Wrapper.MT4Expert for more details).
